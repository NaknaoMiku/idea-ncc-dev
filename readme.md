# 适用于在IntelliJ IDEA上开发用友NCC的插件

### 功能

1. 在一个功能中可以创建多个NCC模块，用友的eclipse版本插件一个项目是一个NCC模块，这样会导致导出的补丁模块不对，该插件可以将不同模块的代码一同打补丁。
2. 复制鉴权文件到Home。
3. 复制upm文件到Home。
4. 复制*.properties资源文件到Home。
5. OpenApi接口测试功能。
